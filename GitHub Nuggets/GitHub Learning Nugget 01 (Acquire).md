# 🎉 My Practice Course

## 👋 Welcome

## 🧠 Quiz Time
What platform are we using for hosting?

[(x)] GitHub

✅ Correct! GitHub is our hosting platform.
</div>

## 🎊 Congratulations!
You've successfully created your first course!
